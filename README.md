 ### 1. Постановка задачи

> Разработать программу для работы с графами, представленными в виде набора рёбер.
Реализовать функции для анализа структуры графа, добавления и удаления рёбер и
вершин, определения свойств графа и выполнения операций над ним.

### 2. Описание функций

1. `toString`: функция выводит строку, содержащию все ребра графа в порядке их добавления.
2. `add`: функция добавляет в массив `lst` ребро и увеличивает флаговую переменную `size`.
3. `printVertexes`: функция создает множество `vertexes` добавляя туда все вершины, затем переделывая в массив `result` сортирует по убыванию, впоследствии заполняя строку вывода.
4. `VertexesMoreN`: функция создает словарь `slov`, заполняет его типом вершина - кол. вхождений, далее проходится по словарю и добавляет в `tempResult` вершины, удовлетворяющие условию, выводит строку.
5. `deleteReb`: функция находит позицию ребра, то после этого все элементы в массиве `lst` после него передвигаются на одну позицию влево, `size1` уменьшается на 1, выводит `true` при удачном удалении.
6. `deleteVer`: функция проходит по всему массиву `lst` если находит совпадение с переданным числом, то удаляет по прицпипу описанном в `deleteReb`.
7. `deleteMinVer`: функция создает словарь по принципу в `VertexesMoreN` далее проходит по нему сравнивая элементы, при этом храня максимум и индекс, передает индекс масимальной вершины в `deleteVer`.
8. `searchComp`: функция создает словарь `groups` типа: вершина - множество его соседей, заполняет его, по принципу если есть вершина, то дабавляет в ее множество другую, если нет то создает новую строку, потом проходит по всем вершинам и группирует их с помошью DFS, выводит массив массивов.
